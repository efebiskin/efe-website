02

Built for the panel.

The Xeneon Edge is a 2560×720 touchscreen that docks to your main display. iCUE — Corsair's control-panel software — lets you drop arbitrary iframes into the widget grid. That's the whole surface this project targets.

Every widget here is one HTML file. No bundler, no JSX, no node_modules. Styles are inline, state is localStorage, network calls are fetch(). That makes them trivially cacheable, portable to any static host, and inspectable without tooling.

Two widgets talk to Meteora, my FastAPI project that wraps Open-Meteo and Yahoo Finance behind a single authenticated gateway with rate limits, tiers, and an in-memory TTL cache.

Honest notes on what doesn't fit

Earlier prototypes included a Discord control widget (mute/deafen/screenshare toggle) and an Apple Music / HomePod widget. Both were removed in v0.2.0 — not because they weren't working, but because a browser tab fundamentally cannot reach the Discord desktop client, and Discord locks voice-control OAuth scopes to partner apps only.

HomePod has no web-callable control API. The only real path is Home Assistant on a LAN device with HomeKit Controller. That's a separate project.

What's here is what fully works from an iFrame — nothing half-built.
